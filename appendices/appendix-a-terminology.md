# 附录 A：术语对照表与全书符号索引

## 一、命名规范概要

本书遵循**"工程用中文，对象用英文；首次双语，复现单语；标题中文，代码英文"**的命名约定。
全文共规范四个核心术语的使用。

### 1.1 四工程术语对照

| 中文（主） | 英文（锚） | 简写（图表/代码/路径） |
|-----------|-----------|----------------------|
| 提示词工程 | Prompt Engineering | Prompt / PE |
| 上下文工程 | Context Engineering | Context / CE |
| 驾驭工程 | Harness Engineering | Harness / HE |
| 循环工程 | Loop Engineering | Loop / LE |

### 1.2 五条使用规则

**规则 1（首次定义）**：章内首次出现用 `中文（English）` 形式。
→ ✅ `提示词工程（Prompt Engineering）`

**规则 2（章内复现）**：第二次起用纯中文。
→ ✅ `提示词工程的目标是...`

**规则 3（层名/范畴）**：指代对象层级时用英文 + 中文层级名。
→ ✅ `Prompt 层`、`Context 工程师`

**规则 4（独立名词性）**：指代操作对象本身时用英文裸名。
→ ✅ `一条 Prompt`、`Harness 实现`

**规则 5（标题/图表/代码）**：章节标题纯中文，代码/路径纯英文。
